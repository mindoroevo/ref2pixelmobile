# Ref2Pixel Mobile

Dieses Repository dient nur dazu, die Datenschutzerklaerung fuer Ref2Pixel ueber GitHub Pages bereitzustellen.

## Inhalte

- `privacy-policy.html` enthaelt die zweisprachige Datenschutzerklaerung in Englisch und Deutsch.
- `index.html` leitet automatisch auf `privacy-policy.html` weiter.

## Zweck

GitHub Pages benoetigt eine Startdatei. Deshalb ist `index.html` nur als Redirect angelegt, damit Aufrufe der GitHub-Pages-URL direkt bei der Datenschutzerklaerung landen.

## Lokal testen

Die Dateien koennen direkt im Browser geoeffnet werden. Beim Oeffnen von `index.html` sollte sofort eine Weiterleitung zur Datenschutzerklaerung erfolgen.

## Deployment

Nach dem Veroeffentlichen ueber GitHub Pages zeigt die Projekt-URL direkt auf die Datenschutzerklaerung.
